This element defines the base metadata content requirements for describing CINERGI inventory items. <br /></p><p><br /></p><p>ENVRI RM v1.1 definition of Metadata: "Data about data, in scientific applications is used to describe, explain, locate, or make it easier to retrieve, use, or manage an information resource. "... "Metadata is generally encoded in a metadata schema which defines a set of metadata elements and the rules governing the use of metadata elements to describe a resource. " ... "Metadata are data, and data become metadata when they are used to describe other data. ... The set of circumstances, purposes, or perspectives for which some data are used as metadata is called the ‘context’. So metadata are data about data in some ‘context’. "  "there [may be] more than one set of metadata related to a single data resource, e.g., when the existing metadata becomes insufficient, users may design new templates to make another metadata collection. "<br /></p><p>"<b>metadata state </b><br /></p><p>--<b>raw</b>: are established metadata, which are not yet registered. In general, they are not shareable in this status <br /></p><p>--<b>registered</b>: are metadata which are inserted into a metadata catalogue. <br /></p><p>--<b>published</b>: are metadata made available to the public, the outside world.
