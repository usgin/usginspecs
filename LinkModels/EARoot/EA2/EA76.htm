Payload represents the information transferred as the content of an HTTP request or response messages. A Payload consists of optional meta information (encoded in HTTP headers) and optional content (encoded in the HTTP body). <br /></p><p><br /></p><p>Furthermore, in API Blueprint context, a payload include its description, description of its message-body attributes and a message-body validation schema.<br /></p><p><br /></p><p>A payload may have its media type associated. A payload's media type represents the metadata received or sent in the form of a HTTP Content-Type header. When specified a payload should include nested Body section.<br /></p>
