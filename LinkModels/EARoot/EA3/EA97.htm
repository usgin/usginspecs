Definition: A collection of data, published or curated by a single agent, and available for access or download in one or more formats.<br /></p><p><br /></p><p>Usage note: This class represents the actual dataset as published by the dataset publisher. In cases where a distinction between the actual dataset and its entry in the catalog is necessary (because metadata such as modification date and maintainer might differ), the catalog record class can be used for the latter.<br /></p>
