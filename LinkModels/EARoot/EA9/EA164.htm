The Agent class is the class of agents; things that do stuff. A well known sub-class is Person, representing people. Other kinds of agents include Organization and Group.<br /></p><p><br /></p><p>The Agent class is useful in a few places in FOAF where Person would have been overly specific. For example, the IM chat ID properties such as jabberID are typically associated with people, but sometimes belong to software bots.<br /></p><p><br /></p><p>http://xmlns.com/foaf/spec/#term_Agent<br /></p><p><br /></p><p>Has Subclass Group Person Organization<br /></p><p><br /></p><p>Properties include: gender yahooChatID account birthday icqChatID aimChatID jabberID made mbox interest tipjar skypeID topic_interest age mbox_sha1sum status msnChatID openid holdsAccount weblog<br /></p>
